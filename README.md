# MVP - IDH e Renda (Estados do Brasil, 2010)

Este repositório contém o código e as evidências de um MVP de dados desenvolvido no Databricks (Community Edition).
O objetivo é analisar a relação entre a renda média domiciliar per capita e o IDH dos estados brasileiros em 2010.

## Fontes de dados
- IDH 2010 (Atlas do Desenvolvimento Humano - PNUD/IPEA/FJP): http://www.atlasbrasil.org.br/consulta/planilha
- Renda média domiciliar per capita 2010 (DATASUS/IBGE): http://tabnet.datasus.gov.br/cgi/tabcgi.exe?ibge/censo/cnv/rendauf

## Estrutura do repositório
- `sql/`: scripts SQL de carga, tratamento, modelagem e análises
- `notebook/`: export do notebook do Databricks
- `evidencias/`: screenshots/vídeos comprovando execução e resultados no Databricks
